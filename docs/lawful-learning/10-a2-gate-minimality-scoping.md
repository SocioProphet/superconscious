## 1. A2 setup

[M] The A2 singularity is the simple isolated hypersurface singularity with local normal form:

```text
f = x0^3 + x1^2 + ... + xk^2
```

[M] Its Milnor number is 2, and its Milnor lattice has rank 2 with A2 Cartan form.

[M] The monodromy is generated by the Coxeter element of `W(A2) = S3`, with order 3.

[T] The A2 loop-class structure therefore replaces the A1 `Z/2` structure with `Z/3`.

## 2. Candidate spatial group

[S|T] The candidate spatial-symmetry group is:

```text
G_A2 = PSU(3)
```

[M] `PSU(3) = SU(3) / Z(SU(3))`, and `Z(SU(3)) = Z/3`.

[M] `PSU(3)` is connected, compact, Lie, and has universal cover `SU(3)`.

[T] The analog of the A1 spatial triad is not literally a triad. For A2, the natural faithful linear frame is the adjoint frame:

```text
su(3) ~= R^8
```

[G] General Lawful Learning documents should use `adjoint frame` or `spatial G_A_n-frame`, not `spatial triad`, outside A1.

## 3. Auxiliary structure

[S|T] The auxiliary group is candidate `SU(3)`, acting on:

```text
V_A2 = C^3
```

via the defining representation.

[M] The central element is:

```text

```

[T] This replaces the A1 central element `-I_2`.

## 4. Form type shift

[M] The defining representation of `SU(2)` is quaternionic type and carries an invariant symplectic form.

[M] The defining representation of `SU(3)` is complex type and does not carry a nonzero invariant bilinear form of the A1 kind.

[T] The A2 polarization form is Hermitian, not symplectic.

[G] Any harness or doctrine artifact that copies A1 symplectic-pairing language into A2 without revision is invalid for A2 reuse.

## 7. Proof obligations

The proof outline has five obligations.

### 7.1 Candidate reduction

[S] Show that connected compact non-abelian candidates carrying the required order-3 loop class reduce to `PSU(3)` under the declared admissibility conditions.

### 7.2 Admissibility check

[M] Verify the standard structural facts:

```text
PSU(3) connected compact Lie
pi_1(PSU(3)) = Z/3
SU(3) -> PSU(3) universal cover
Ad_PSU(3) faithful on su(3)
defining SU(3) action preserves H_std
```

### 7.3 Auxiliary necessity

[S|M] Establish that `PSU(3)` has no faithful 3-dimensional complex linear representation, so the `C^3` action must live in the auxiliary `SU(3)` structure and descend only projectively.

### 7.4 Schur uniqueness

[S|M] Replace the A1 symplectic scalar constraint with the A2 Hermitian-plus-determinant constraint:

```text
|lambda| = 1
lambda^3 = 1
lambda in {1, omega, omega^2}
```

Then use the coherent loop condition to select the canonical branch.

### 7.5 Minimality

[S] Once candidate reduction and auxiliary uniqueness are complete, minimality follows by the same structural shape as A1.
