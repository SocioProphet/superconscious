# Neuro-Symbolic Cognition Position

Status: v0.1 doctrine capture.

This document aligns the CHRONOS / ASU neuro-symbolic reasoning capture with the Superconscious cognition loop.

It is intentionally a boundary document. It does not add live neuro-symbolic execution, model calls, network calls, tool calls, schema promotion, memory promotion, ontology promotion, or policy admission.

## Position

Superconscious is the visible governed cognition loop over neuro-symbolic capabilities.

It may coordinate, classify, request, record, emit safe traces, propose remediation, emit risk signals, and request policy review.

It must not become the authority for symbolic truth, schema, ontology, policy admission, execution, runtime placement, model routing, model governance, memory promotion, or workspace topology.

## Relationship to CHRONOS

CHRONOS is the governed evidence loop.

Superconscious is the cognition-loop surface that can invoke or record CHRONOS lifecycle events:

```text
Task / observation
  -> neuro-symbolic capability classification
  -> evidence / carrier request
  -> reflection
  -> remediation proposal
  -> risk signal
  -> policy review request
  -> AgentPlane-compatible evidence reference
```

The upstream SocioSphere CHRONOS capture defines the evidence-loop integration doctrine. Superconscious consumes that doctrine operationally, but does not own corpus-loop authority.

## Allowed Superconscious roles

Superconscious may:

- classify a neuro-symbolic method family;
- request a CHRONOS carrier evaluation;
- record evidence references and method references;
- emit a safe operational trace;
- propose remediation through AgentPlane;
- record learning-loop evidence;
- emit a risk signal;
- request policy review;
- request model routing through the routing authority;
- request schema validation through the schema authority;
- request ontology vocabulary review through Ontogenesis.

## Forbidden Superconscious roles

Superconscious must not:

- treat a fuzzy score as truth;
- treat neural output as evidence without provenance;
- treat a learned rule as canonical schema;
- treat symbolic derivation as policy admission;
- promote ontology embeddings as ontology authority;
- persist durable memory from unadmitted carriers;
- execute a symbolic policy as a controller;
- open network egress for an NSR method in M1 / M1.5 posture;
- call model providers directly;
- bypass AgentPlane for execution, evidence, or replay;
- bypass Policy Fabric for admission or cancellation.

## Neuro-symbolic method families

Superconscious recognizes these method families as integration roles:

| Family | Superconscious role | Boundary |
|---|---|---|
| `NSR-FOUNDATION-LOGIC` | vocabulary reference for atoms, rules, grounding, entailment, annotated truth, and fixpoint closure | not theorem-grade unless proof system is explicit |
| `NSR-TAXONOMY` | classify method family | not an authority or maturity grade |
| `NSR-SOFT-CONSTRAINT` | record semantic satisfaction / fuzzy score | advisory only |
| `NSR-TRUTH-BOUND` | record truth-bound or inconsistency report | local diagnostic only |
| `NSR-SYMBOLIC-ADJUDICATION` | request symbolic adjudication over neural observations | still requires provenance and policy admission |
| `NSR-DIFFERENTIABLE-CONSTRAINT-LEARNING` | record candidate constraint and grounding risk | requires leakage and transduction validation |
| `NSR-RULE-LEARNING` | propose learned rule candidate | never canonical without owning-plane review |
| `NSR-ONTOLOGY-INFERENCE` | propose ontology delta candidate | embeddings are not ontology authority |
| `NSR-SYMBOLIC-POLICY` | propose symbolic policy candidate | not live controller until admitted |

## Safe trace requirement

Any Superconscious trace referencing neuro-symbolic work must include:

- method family;
- upstream source or evidence reference;
- carrier status;
- provenance status;
- validation status;
- authority plane;
- governance state;
- forbidden promotion note;
- replay reference or pending replay reference.

## Non-claim boundary

This document does not claim that neuro-symbolic systems solve consciousness, reasoning, truth, policy, or ontology authority.

It claims only that Superconscious can make neuro-symbolic cognition inspectable, typed, replayable, and policy-bound when the authority boundaries are explicit.
