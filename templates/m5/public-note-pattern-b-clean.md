# Feature steering crosses the boundary of the natural envelope

## Abstract

This note reports a boundary-crossing implementability result for refuse-and-redirect feature steering. The behavioral effect is real, but the activation profile producing it approaches or crosses the declared natural-envelope boundary.

## One conceptual move

Feature steering needs an implementability test.

## Load-bearing figure

Insert the five-curve plot showing the studied feature crossing the q95/q99 region near `alpha_behavioral`.

## Result frame

Pattern B: feature steering is behaviorally effective but partly extrapolative.

## Interpretation

The result complicates ordinary steering claims. A continuous behavioral improvement curve does not imply a continuous in-distribution activation path.

## Limitations

No cross-model generalization is claimed. Transcoder evidence is deferred. Safety conclusions depend on the M1D off-target audit outcome.
